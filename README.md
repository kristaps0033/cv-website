# CV Website (Cloudflare Pages)

## Saturs
- `index.html` – galvenā lapa
- `styles.css` – stili
- `assets/` – attēli (ievieto ekrānuzņēmumus no mapes "Bildes")

## Ātra publicēšana ar Cloudflare Pages (Direct Upload)
1. Atver Cloudflare Dashboard → Pages → Create a project → Upload assets.
2. Izvēlies mapi `cv-website` kā projekta sakni.
3. Publicē.

## Attēli
Ja vēlies redzēt ekrānuzņēmumus:
1. Nokopē attēlus no `homelab-portal/Bildes` uz `cv-website/assets`.
2. Pārliecinies, ka failu nosaukumi sakrīt ar tiem, kas norādīti `index.html`.

## Domēns
Pēc publicēšanas Cloudflare Pages dos pagaidu URL. Domēnu (piem., `cv.kristapshomelab.com`) var piesaistīt Pages → Custom domains.
